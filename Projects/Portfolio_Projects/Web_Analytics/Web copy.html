Back Story

This was an independent project done as part of a Web Analytics initiative. The project focused on understanding why digital products were underperforming and how structured, data-driven testing could improve their performance across the user journey.

Objective of the Project

To identify funnel-level gaps using structured frameworks and apply statistically validated experiments to improve adoption, engagement, and long-term retention.

Tools and Techniques Used

  • Google Analytics
  • A/B testing
  • Statistical hypothesis testing
  • Likert scale based surveys
  • Excel based statistical analysis

Tests Conducted

1. Ad Messaging A/B Test

What this test did: It evaluated the impact of different ad copies on Click Through Rate (CTR) by comparing a generic message with a benefit-focused message.

Result: CTR improved from 11% to 27%, showing strong statistical uplift.

2. Channel Effectiveness Test

What this test did: It measured Engagement Rate across multiple platforms including social media, email, and display ads to identify the most effective channel.

Result:Social media showed 14% engagement, email 9%, and display ads 4%.

3. Brand Alignment Survey Test

What this test did: It evaluated how well the product value proposition resonated with users using a 5-point Likert Scale.

Result:The Brand Alignment Score improved from 2.8 to 3.9.

4. Message Clarity Test

What this test did: It measured Message Comprehension Rate by comparing original onboarding content with simplified content.

Result: Comprehension improved from 28% to 72%.

5. Setup Time Retention Analysis

What this test did: It analyzed how setup time correlated with retention.

Result:Users with setup time under 6 minutes showed 79% retention; those above 6 minutes showed only 18%.

6. Net Promoter Score (NPS) Test

What this test did: Measured overall user satisfaction through NPS.

Result:NPS increased from 12 to 41.

Metrics Tracked

  • Click Through Rate (CTR)
  • Bounce Rate
  • Engagement Rate
  • Brand Alignment Score
  • Message Comprehension Rate
  • Retention Rate
  • Net Promoter Score (NPS)

Output

The overall output of the project showed statistically significant improvements across the funnel. Key performance metrics improved consistently after each test, proving that structured experimentation and data-driven decisions can strongly enhance digital product performance.
