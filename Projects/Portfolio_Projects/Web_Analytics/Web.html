This project focused on identifying gaps in the digital product funnel and improving adoption, engagement, and retention through structured, data-driven experiments. The project leveraged tools like Google Analytics, A/B testing, statistical hypothesis testing, Likert scale surveys, and Excel for data analysis to evaluate ad messaging, channel effectiveness, brand alignment, message clarity, setup time retention, and user satisfaction. The output was a set of validated insights that significantly improved key metrics such as CTR, engagement, comprehension, retention, and NPS. Through this project, I learned the importance of designing rigorous experiments, handling cross-platform data, and combining technical analysis with business-facing communication to drive actionable improvements.

This was an independent project done as part of a Web Analytics initiative. The project focused on understanding why digital products were underperforming and how structured, data-driven testing could improve their performance across the user journey.

To identify funnel-level gaps using structured frameworks and apply statistically validated experiments to improve adoption, engagement, and long-term retention.

  • Google Analytics
  • A/B testing
  • Statistical hypothesis testing
  • Likert scale based surveys
  • Excel based statistical analysis

1. Ad Messaging A/B Test
Evaluated the impact of different ad copies on CTR. Result: CTR improved from 11% to 27%.

2. Channel Effectiveness Test
Measured engagement across platforms. Social media: 14%, email: 9%, display ads: 4%.

3. Brand Alignment Survey Test
Evaluated value proposition resonance. Score improved from 2.8 to 3.9.

4. Message Clarity Test
Measured comprehension improvement: 28% → 72%.

5. Setup Time Retention Analysis
Setup < 6 min: 79% retention; > 6 min: 18%.

6. Net Promoter Score (NPS) Test
NPS increased from 12 to 41.

The project showed statistically significant improvements across the funnel, proving that structured experimentation and data-driven decisions can strongly enhance digital product performance.

  • Conducted A/B Testing that improved simulated ad CTR by 2.6x.
  • Simulated RCT to reduce product page bounce rate by 30%.
  • Applied Starch Testing to reveal a 24% awareness gap.
